## **⚙️ Clash Verge 深度配置指引**

安装完软件后，请按照以下步骤完成配置：

### **第一步：在“订阅”页粘贴自己的订阅链接，然后点击导入。**

### **第二步：选择代理模式与节点**

1. 点击左侧的 **“代理 (Proxies)”**。  
2. 在上方选择 **“规则模式 (Rule)”**（强烈建议：此模式下国内流量不走代理，速度最快）。  
3. 在节点列表中，点击展开 **“代理选择”** 或 **“手动切换”**，测速后选择一个延迟较低的节点。

### **第三步：开启系统代理与分流设置**

1. 在订阅页的**“全局订阅扩展配置”中“编辑文件”，将github.com/Loyalsoldier中的分流规则导入**。  
2. 在首页打开 **“系统代理 (System Proxy)”** 选项。  
3. 开启 **“开机自启动 (Start on Login)”**，确保每次开机即用。
